# ansible-eda-debug
Rulebook and playbook to  test the EDA-->Automation Execution communication

1. Create a project that points to this Git Repo Under Automation Decisions 
2. Create a rulebook activation with the following parameters set:
   * Decision Environment: Default Decision Environemnt
   * Credential: AAP Credential pointing towards controller/AAP Gateway that is to be used for the action and testing
   * Restart Policy: Never
   * Log Level: Debug
